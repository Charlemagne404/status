How this page handles data

Operational data

The page requests status information from the local `/api/status` endpoint to display service availability.

Server logs

Standard web server logs may include IP address, request path, user agent, and timestamp for security, troubleshooting, and uptime monitoring.

Cookies and tracking

This site does not set application cookies and does not include third-party analytics scripts by default.
